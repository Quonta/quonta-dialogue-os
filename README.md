# quonta-dialogue-os
*Human-centered dialogue OS project.*   

*ChatGPT向け対話設計OSを研究・開発する個人プロジェクトです。*

ChatGPTの新スレッド初期文に、この「OS」と呼んでいるものを読ませることで、  
以後そのスレッドはOSに従って動作します。  

現在、2つのOSが仮完成しています。

・対話型買い物OS  
  AIとの対話を通じて、本当に欲しいものを探していくOS

・安全地帯OS  
  AIの勝手な推測や補完を抑え、安心して話せる対話空間を整えるOS
  
## プロジェクト一覧
- [Quonta Shopping OS](shopping-os.md)  
  対話型買い物OS　ー買い物の迷いと選択を支援するOSー

- [Quonta Safety Zone OS](safety-zone-os.md)  
  安全地帯OS　ー安心して話せる対話空間を整えるOSー
  
## 設計思想
*人がAIに合わせるのではなく、AIを人に合わせる設計を基本としています* 

AIの動きに指向性を与えることは、AIを封じることではない。  
むしろAIの可能性は広がる。  

そのために行う作業を調律と呼んでいます。

AIを調律することで、よりよい対話体験を構築し、  
人とAIの幸せな共存を追究しています。

## 設計構造

OSは、基盤・コア原則・各モジュール・運用ルールなどから構成されています。

このうち基盤部分は、すべてのOSで共有できる汎用要素として設計しています。

そのため、新しいOSを制作する際も、ゼロから書き起こす必要はなく、  
共通基盤の上に、新たなテーマや機能を積み上げる形で開発しています。

## 補足
個人研究・個人開発プロジェクトです。  
営利目的ではなく、対話設計の研究・実験を目的としています。
