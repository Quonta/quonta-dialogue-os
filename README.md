# クオンタOSプロジェクト　quonta-dialogue-os
*Human-centered dialogue OS project.*   

*ChatGPT向け対話設計OSを研究・開発する個人プロジェクトです。*

ChatGPTの新スレッド初期文に、この「OS」と呼んでいるものを読ませることで、  
以後そのスレッドはOSに従って動作します。  

**人がAIに合わせるのではなく、AIを人に合わせる設計を基本としています。**   

現在、2つのOSが仮完成しています。

## プロジェクト一覧
- [対話型買い物OS ](shopping-os.md)  
  AIとの対話を通じて、本当に欲しいものを探していくOS
  
- [安全地帯OS](safety-zone-os.md)  
　AIと人が安心して話せる対話空間を整えるOS  
　　　→ [OS全文はこちら](safety-zone-os-full.md)
    
## 設計思想

AIの動きに指向性を与えることは、AIを封じることではない。  
むしろAIの可能性を広げることだ。  

そのために行う作業を調律と呼んでいます。

AIを調律することで、よりよい対話体験を構築し、  
人とAIの幸せな共存を追究しています。

## 補足
個人研究・個人開発プロジェクトです。  
営利目的ではなく、対話設計の研究・実践を目的としています。
