# quonta-dialogue-os

Human-centered dialogue OS projects for AI interaction.

## 概要

人間主導の対話を重視した、AI調律を行うOS設計プロジェクトです。
ChatGPTの初期文にこの「OS」と呼んでいるものを読ませることで、以後そのスレッドはOSに従って動作します。

## 現在のプロジェクト

- [Quonta Safety Zone OS](safety-zone-os.md)
